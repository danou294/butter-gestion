# Répertoire pour les GIFs et Memes de chargement

Ce répertoire contient les GIFs et images (PNG/JPEG) qui s'affichent pendant le chargement de l'import.

## Formats acceptés
- `.gif` - GIFs animés
- `.png` - Images PNG
- `.jpeg` / `.jpg` - Images JPEG

## Utilisation
Placez vos fichiers dans ce répertoire. Ils seront automatiquement affichés de manière aléatoire pendant le chargement de l'import.

## Exemple de fichiers
- `loading1.gif`
- `meme1.png`
- `funny.gif`
- etc.

Les fichiers seront servis via Django static files.
